# DATA SETUP

1. Clone the project
2. Install: `uv sync` (use uv, not pip)
3. Download data from: https://drive.google.com/drive/folders/1zDvr6G3U6g0ZLbcnqvgt7TJD84VaNS6w?usp=sharing
4. Extract to `processed/` folder
5. Work in your component folder
6. Don't commit data files (.csv, .xlsx) to GitHub

Data includes: 10 transformers, 12 months, locations, solar info, consumption data.
